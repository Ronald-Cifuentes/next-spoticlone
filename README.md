### Run Local:

```
git clone https://github.com/R0N4LDC1FU3NT3S/calculator.git
```

create an .env file and add this variables

```bash
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
```

```bash
$ npm Install
$ npm run dev
```
